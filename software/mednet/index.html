Mednet: Computer-Aided Disease Detection and Grading from Medical Images

Posted on Fri 01 September 2023 in software

Mednet is an open-source software package that can train and evaluate convolutional neural network (CNN) architectures to detect and grade diseases based on medical images. Examples include Pulmonary Tuberculosis (TB) detection based on chest X-rays (CXR) and Inflammatory Eye-Diseases based on fundus angiograms.
